# CLIMAFORM

A MINIMALIST WEATHER FORECAST APPLICATION WITH A FOCUS ON CLEAN DESIGN AND REAL-TIME DATA.

---

### CORE FEATURES

-   REAL-TIME WEATHER DATA (TEMPERATURE, WIND, HUMIDITY)
-   EXPANDABLE 3-DAY FORECAST
-   GEOLOCATION ON LOAD
-   SEARCH BY CITY
-   UNIT CONVERSION (°C / °F)

---

### TECH STACK

-   `REACT`
-   `VITE`
-   `TYPESCRIPT`
-   `TAILWIND CSS`
-   `FRAMER MOTION`
